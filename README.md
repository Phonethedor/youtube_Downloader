## Youtube Downloader

### Como funciona

1. Genera un PO Token y visitorData para evadir el bloqueo 403 / restricción a 360p de YouTube
2. Obtiene la información del video
3. Descarga el video

### Como ejecutar

```bash
# Instalar dependencias
npm install

# Ejecutar la app
npm start
```

### Como compilar

```bash
# Compilar la app
npm run dist
```

### Como empaquetar

```bash
# Empaquetar la app
npm run package
```

### Como limpiar

```bash
# Limpiar la app
npm run clean
```

### Como desinstalar
